Please add the following images to this folder:

1. home.png - Home icon for dashboard
2. settings.png - Settings icon
3. categories.png - Categories icon
4. menu.png - Menu items icon
5. logout.png - Logout icon
6. categories-card.png - Large categories card image
7. menu-card.png - Large menu card image
8. settings-card.png - Large settings card image

Recommended image sizes:
- Sidebar icons (home.png, settings.png, etc): 24x24 pixels
- Card images (*-card.png): 80x80 pixels

All images should be in PNG format with transparent backgrounds.
